## ðŸŽ¯ Motivation

The rise in AI-generated materials threatens the pillars of Information Assurance: **Integrity**, **authenticity**, and **non-repudiation**. In academic and professional domains, forged documents like essays, resumes, and certificates compromise reliable decision-making. Detecting such content restricts the malicious use of Generative AI, ensures fairness and authenticity in academic evaluation and professional recruitment, and preserves ethical standards.

---

## Scope of this Project

This project will focus on the following objectives:

* Detect AI-generated academic documents (e.g., essays, research papers).
* Identify forged professional documents (e.g., resumes, certificates, experience letters).
* Compare tools and techniques of non-emerging approaches that provide baseline comparisons, such as:
    * Cryptographic watermarking
    * Stylometry
    * Metadata analysis
* Assess the detection of AI-generated images and text using emerging technologies, including:
    * CNN-based image forensics
    * Watermarking
    * Vision-based approaches
    * Feature-based detection
    * Adversarial learning
* Explore applications in academic plagiarism detection systems, applicant tracking systems (ATS), and HR verification platforms.
